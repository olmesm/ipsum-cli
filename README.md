# Ipsum cli

Copy Lorem Ipsum text into your clipboard from the cli.

Defaults to 3 paragraphs, but accepts an optional number argument between 1-20.

```bash
# Running via npx
npx ipsum-cli <qty>

# Install Globally
npm install -g ipsum-cli
ipsum <qty>

# Uninstall
npm uninstall -g ipsum-cli
```

## Developing

```bash
node . <qty>

# Test Globally
npm install -g .
ipsum <qty>

# Uninstall
npm uninstall -g ipsum-cli
```
